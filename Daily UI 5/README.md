# Daily UI #005 - App Icon Grid

A modern and vibrant set of 9 app icons created for the Daily UI challenge. This project showcases the power of hand-coded SVGs and CSS gradients to create a cohesive and visually appealing icon set.

## 📱 Featured Icons
The grid includes icons for common utility apps:
- **Folder**: Vibrant yellow-orange gradient with a classic folder shape.
- **Calendar**: Professional dark blue-teal gradient with a minimalist date view.
- **Profile**: Elegant purple-indigo gradient with a clean user silhouette.
- **Contacts**: Bright green-lime gradient with a simplified person icon.
- **Messages**: Deep teal-blue gradient with a sleek chat bubble.
- **Photos**: Warm yellow-red gradient with a abstract landscape icon.
- **Videos**: Stylish deep purple-pink gradient with a play button.
- **Location**: Fresh seafoam-emerald gradient with a location pin.
- **Documents**: Rich orange-yellow gradient with a lined paper icon.

## 🛠️ Design & Implementation
- **SVG Mastery**: Every icon is constructed using SVG elements like `<rect>`, `<circle>`, `<path>`, and `<polygon>`.
- **Dynamic Gradients**: Each icon uses a custom `<linearGradient>` defined within its own SVG scope for unique color transitions.
- **Responsive Layout**: Designed with a clean, centered grid that adapts to different screen sizes.
- **Modern Aesthetics**: Utilizes semi-transparent overlays and soft rounded corners (28px radius) to achieve a modern, "glassmorphism-lite" feel.

## 🚀 How to View
Simply open `index.html` in any modern web browser to view the icon grid.
