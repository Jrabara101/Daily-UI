# Daily UI #006 - User Profile Page

A professional, interactive user profile page designed to showcase personal statistics, social connections, and a real-time activity timeline. This project emphasizes a clean UI, smooth interactions, and personalization features.

## ✨ Key Features

- **Interactive Profile Card**: Displays user identity with an editable name and bio.
- **Dynamic Stats & Badges**: Showcases engagement metrics (Posts, Followers, Following) and achievement badges (Streak, Top Coder, Level).
- **Real-time Activity Timeline**: A functional feed where users can publish new updates instantly without page reloads.
- **Dark Mode Support**: A sleek dark theme toggle that persists via CSS variables for a comfortable viewing experience.
- **Profile Customization**: 
  - **In-place Avatar Upload**: Change the profile picture with an immediate preview that syncs across the whole UI.
  - **Edit Modal**: A dedicated interface for updating personal information seamlessly.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.

## 🛠️ Technical Stack

- **HTML5**: Semantic markup for accessibility and structure.
- **CSS3 (Vanilla)**: 
  - Custom CSS variables for theme management.
  - Flexbox and Grid for layout precision.
  - Smooth transitions and hover effects for premium feel.
- **JavaScript (ES6+)**: 
  - DOM manipulation for the timeline and profile updates.
  - File Reader API for local image previews.
  - Event-driven UI updates.
- **Iconography**: Powered by **Font Awesome 6** for high-quality, scalable graphics.

## 🚀 Getting Started

1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. Use the **Dark Mode** toggle in the top right.
4. Click **Edit Profile** to change your details.
5. Post an update in the **Timeline** to see it appear live.
