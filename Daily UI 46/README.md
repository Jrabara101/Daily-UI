# Daily UI #46 - Service Invoice

A high-end, formal service invoice template designed for professional business use. Built with pure HTML and CSS, optimized for both screen display and A4 printing.

### Professional Typography
- **Headings**: Playfair Display (serif) for elegant, formal appearance
- **Body Text**: Inter (sans-serif) for optimal readability
- Careful hierarchy with appropriate font sizes and weights

### Layout & Structure
- **Header Section**: Company logo placeholder, company name, and invoice metadata
- **Two-Column Address Grid**: "Bill To" and "Ship To" sections with contact information
- **Detailed Services Table**: Itemized services with descriptions, quantities, rates, tax, and totals
- **Totals Section**: Clear breakdown of subtotals, discounts, tax, and grand total
- **Payment Methods**: Multiple payment options with detailed instructions
- **Footer**: Payment terms, thank you message, and complete company contact information

6. **Multiple Payment Methods**

   - Payment terms and conditions

## üìã Customization

To customize this template:

1. **Company Information**: Update company name, tagline, and contact details in the header and footer
2. **Logo**: Replace the SVG logo placeholder with your company logo
3. **Colors**: Modify CSS custom properties (variables) in `:root` selector
4. **Services**: Update the services table with your actual service items
5. **Payment Details**: Update bank account and payment portal information
6. **Fonts**: Change font families in the `:root` CSS variables

## üöÄ Usage

1. Open `index.html` in a web browser
2. Customize with your company information
3. Print to PDF using browser's print function (Ctrl+P / Cmd+P)
4. Select "Save as PDF" for digital distribution
5. Or print directly to A4 paper
