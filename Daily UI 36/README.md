## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Components

- **Navbar**: Fixed navigation with cart counter
- **SpecialOfferHero**: Main hero section with parallax effects and promo code
- **PromoCodeCard**: Interactive promo code display with copy functionality
- **Timer**: Countdown timer component
- **OfferDetails**: Feature highlights section
