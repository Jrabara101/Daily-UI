# Daily UI 009: Music Player

A sleek and interactive music player interface built with vanilla HTML, CSS, and JavaScript. This project was developed as part of the Daily UI challenge, focusing on creating a seamless and aesthetically pleasing user experience for audio playback.

## ✨ Features
- **Dark/Light Mode**: Toggle between a deep space dark theme and a clean, soft light theme.
- **Dynamic Playback**: Full support for Play/Pause, Next Track, Previous Track, Shuffle, and Repeat.
- **Interactive UI**:
  - Rotating album art animation during playback.
  - Interactive progress bar for track scrubbing.
  - Volume control slider.
- **Playlist Management**: A scrollable playlist that highlights the currently playing track.
- **Responsive Design**: Fully optimized for mobile and desktop screens.

## 🎨 Design Aesthetics
- **Color Palette**: 
  - Dark Mode: `#0f3460` (Primary), `#16a085` (Accent)
  - Light Mode: `#ffffff` (Primary), `#e8a87c` (Accent)
- **Typography**: Clean and legible **Roboto** font from Google Fonts.
- **Icons**: Industry-standard **Font Awesome 6** icons for intuitive controls.
- **Animations**: Smooth CSS transitions and a continuous spin animation for the album art.

## 🛠️ Built With
- **HTML5**: Semantic structure.
- **CSS3**: Custom layout using Flexbox and CSS variables for theming.
- **JavaScript**: Vanilla JS for audio logic, DOM manipulation, and theme switching.

## 🚀 How to Run
1. Clone this repository.
2. Open `index.html` in any modern web browser.
3. Enjoy your music interface!
