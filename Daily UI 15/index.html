‚ôø Accessibility Features

‚úì WCAG 2.1 Compliant

Our switches are fully accessible and follow web content accessibility guidelines:

  • ARIA Labels: Proper role="switch" and aria-label attributes
  • Keyboard Navigation: Full keyboard support with Space/Enter keys
  • Focus Management: Visible focus indicators for keyboard users
  • Color Independence: Not reliant solely on color to convey meaning
  • Screen Reader Support: Clear state indication for assistive technologies
  • Reduced Motion: Respects prefers-reduced-motion for animations
  • Disabled State: Proper visual and functional disabled state
