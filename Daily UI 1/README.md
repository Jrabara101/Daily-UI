# Daily UI 1: Sign Up Page

This project is a visually engaging Sign-Up page built for the Daily UI challenge (#001). It focuses on creating a premium feel using modern web technologies and micro-interactions.

## Tech Stack
- **Framework:** React 19 + Vite
- **Styling:** Tailwind CSS v4
- **Animation:** GSAP for entry animations
- **Effects:** `@paper-design/shaders-react` for dynamic background shaders

## Features
- **GSAP Entry Animation:** The sign-up form smoothly drops in and fades into view on load.
- **Background Shader:** A lush dynamic background shader wraps the layout to provide depth and an aesthetic feel.
- **ClickSpark Interactions:** Clicking around the form triggers a custom `ClickSpark` visual particle effect for instant user feedback.
- **Form State Management:** Implements basic React state to handle input, submission, and displaying a success message upon completion.

## Running Locally

To run the development server:

```bash
npm install
npm run dev
```
